# 🐱 Feline Friends Adoption 🐱

A cat adoption website built as a final project for the **Sisters of Code Ambassador Program**.

---

## About the Project

Feline Friends is a static multi-page website for a fictional cat adoption shelter. It allows visitors to browse cats available for adoption, learn about the shelter, and get in touch to start the adoption process.

This project was created as a final project submission for [Sisters of Code](https://www.sistersofcode.org), an organization empowering women in tech. It was built as part of the Ambassador Program.

---

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero section, adoption process steps, and success stories |
| Adopt | `adopt.html` | Browse all cats available for adoption |
| About | `about.html` | About the shelter and its mission |
| Contact | `contact.html` | Contact form and important information |

## Future Enhancements

1. **Cat profile modals** — Clicking a cat card opens a popup with full details and an adopt button. Requires JavaScript to show/hide the modal.
2. **Search & filter** — The search bar and filter dropdowns (breed, age, color, etc.) need JavaScript to filter cat cards in real time.
3. **Sort by** — The "Newest Arrivals / Oldest" dropdown needs JavaScript to reorder the cat cards.
4. **Contact form submission** — The form needs JavaScript to validate inputs and send the message.
5. **Navigation highlight** — Highlighting the active nav link needs JavaScript to detect the current page.
6. **Mobile hamburger menu** — On small screens the nav links should collapse into a menu that toggles open/close, requiring JavaScript.
7. **Responsive design** — The site needs to adapt to different screen sizes using CSS media queries.
